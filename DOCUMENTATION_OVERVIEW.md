## The Golden Rules

No matter which file you use, remember these 3 rules:

### Rule 1: Make Your Sheet PUBLIC
- **Most common problem**: Sheet is private
- **Solution**: Share → "Anyone with the link" → "Viewer" access
- **This fixes 50% of all issues!**

### Rule 2: Use EXACT Column Names
- **Most common problem**: Column headers misspelled
- **Solution**: Match exactly (including capitalization)
- **Examples**: "Price" not "price", "WhatsApp" not "whatsapp"

### Rule 3: Use Correct Data Formats
- **Most common problem**: Price has $ or Duration has "min"
- **Solution**: Numbers only (65, not "$65" or "65.00")
- **This fixes 40% of remaining issues!**

These 3 rules solve 90% of all problems!
