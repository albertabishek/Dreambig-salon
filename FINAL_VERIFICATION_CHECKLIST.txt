TROUBLESHOOTING:
================

If something doesn't show:

1. Check Google Sheet is PUBLIC
   → Share → Change to "Anyone with link" → Viewer

2. Check column headers are EXACT (case matters for Day names)
   → Settings: Setting | Value
   → Hours: Day | Hours (day values: monday-sunday lowercase)
   → Services: Name | Price | Duration | Category | Description | Image URL
   → Gallery: Title | Image URL | Description

3. Check data format:
   → Phone: +91-9234567890 (WITH country code)
   → Hours: 10:00 AM - 8:00 PM (12-hour format)
   → Days: lowercase (monday, not Monday)
   → Price: numbers only (65, not $65)
   → Duration: numbers only (60, not 60 min)

4. Hard refresh (Ctrl+Shift+R)

5. Clear browser cache if still not working:
   → Settings → Clear browsing data → All time

6. Check browser console for errors:
   → F12 → Console tab → Look for red errors
