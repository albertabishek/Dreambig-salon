## The 4 Google Sheets

### 1. Settings Sheet
**What:** Salon information (name, phone, email, address, hours, map)
**Used On:** Navbar, Footer, Contact Page, Map
**Columns:** Setting | Value
**Guide:** FINAL_COMPLETE_SETUP.md → Sheet 1

### 2. Services Sheet
**What:** Your salon services with prices and durations
**Used On:** Services Page, Home Preview
**Columns:** Name | Price | Duration | Category | Description | Image URL
**Guide:** FINAL_COMPLETE_SETUP.md → Sheet 2

### 3. Gallery Sheet
**What:** Salon photos and images
**Used On:** Gallery Page, Home Preview
**Columns:** Title | Image URL | Description
**Guide:** FINAL_COMPLETE_SETUP.md → Sheet 3

### 4. Hours Sheet
**What:** Weekly opening hours for all 7 days
**Used On:** Footer, Contact Page
**Columns:** Day | Hours
**Guide:** FINAL_COMPLETE_SETUP.md → Sheet 4 or HOURS_SHEET_SETUP.md

## Critical Rules (Don't Forget!)

```
PHONE FORMAT:
✓ +91-9234567890
✗ 9234567890
✗ 91-9234567890

HOURS DAY NAMES (All lowercase):
✓ monday, tuesday, wednesday, thursday, friday, saturday, sunday
✗ Monday, MONDAY, Mon

HOURS FORMAT (12-hour):
✓ 10:00 AM - 8:00 PM
✗ 10:00 - 20:00

NUMBERS:
✓ Price: 65 (no $ or commas)
✓ Duration: 60 (no min)

URLs:
✓ https://example.com/image.jpg
✗ example.com/image.jpg

SETTINGS:
✓ Sheet is PUBLIC (Anyone with link can View)
✗ Private or restricted access
```

## Troubleshooting by Issue

### Phone/WhatsApp Not Working
1. Check: FINAL_COMPLETE_SETUP.md → Phone Format Section
2. Verify: Format is +CountryCode-Number
3. Test: VERIFY_SETUP_WORKS.md → Phone Testing Section

### Hours Not Showing
1. Check: SHEET_SETUP_VISUAL.txt → Tab 4 Section
2. Verify: Day names are lowercase (monday, not Monday)
3. Test: VERIFY_SETUP_WORKS.md → Hours Verification

### Services/Gallery Not Showing
1. Check: Image URLs start with https://
2. Verify: All required columns filled
3. Test: VERIFY_SETUP_WORKS.md → Services/Gallery Sections

### Map Not Showing
1. Check: FINAL_COMPLETE_SETUP.md → How to Find Latitude/Longitude
2. Verify: Numbers are correct format (28.6139, 77.2090)
3. Test: VERIFY_SETUP_WORKS.md → Map Testing

### Nothing Updated
1. Hard refresh: Ctrl+Shift+R
2. Wait: 1 hour for cache update
3. Check: VERIFY_SETUP_WORKS.md → Debugging Section
