                    GOOGLE SHEETS QUICK REFERENCE

YOUR SALON WEBSITE NEEDS 3 SHEETS:
┌─────────────────────────────────────────────────────────────────────────────┐
│  1. SERVICES        → All salon services with prices & duration              │
│  2. GALLERY         → Photos of your salon                                   │
│  3. SETTINGS        → Salon info (name, phone, address, etc.)               │
└─────────────────────────────────────────────────────────────────────────────┘

================================================================================
                          SHEET 1: SERVICES
================================================================================

COLUMNS (in this EXACT order):
─────────────────────────────────────────────────────────────────────────────
| A      | B     | C        | D        | E           | F           |
| Name   | Price | Duration | Category | Description | Image URL   |
─────────────────────────────────────────────────────────────────────────────

ROW 1 = HEADERS (exactly as shown above)
ROWS 2+ = YOUR DATA

EXAMPLE ROW:
Hair Cutting & Styling | 65 | 60 | Hair | Professional haircuts... | https://...jpg

RULES FOR EACH COLUMN:
───────────────────────────────────────────────────────────────────────────────
A: Name
   - Text (up to 50 characters)
   - Examples: "Hair Cutting & Styling", "Facial Treatment"

B: Price
   - NUMBERS ONLY (no $ sign, no decimal)
   - Examples: 65, 85, 50
   - NOT: $65, 65.00, "$65"

C: Duration
   - NUMBERS ONLY (minutes, no "min" text)
   - Examples: 60, 45, 90
   - NOT: 60 min, 1 hour, "45 mins"

D: Category
   - MUST be EXACTLY ONE of these:
     * Hair
     * Skincare
     * Nails
     * Wellness
     * Makeup
   - Case-sensitive! Use exact capitalization above

E: Description
   - Text (1-3 sentences, 50-150 characters)
   - Describes what the service includes
   - Keep it marketing-friendly

F: Image URL
   - COMPLETE URL starting with https://
   - Examples:
     * https://images.unsplash.com/photo-1522337660859-02fbefca6088?w=400
     * https://images.pexels.com/photos/3407617/pexels-photo-3407617.jpeg
   - If empty, website shows placeholder
   - Image must be publicly accessible

REQUIRED MINIMUM:
  - At least 3 services
  - All columns filled except Image URL (can be empty)

================================================================================
                          SHEET 2: GALLERY
================================================================================

COLUMNS (in this EXACT order):
─────────────────────────────────────────────────────────────────────────────
| A     | B          | C           |
| Title | Image URL  | Description |
─────────────────────────────────────────────────────────────────────────────

ROW 1 = HEADERS
ROWS 2+ = YOUR DATA

EXAMPLE ROW:
Salon Interior | https://images.unsplash.com/photo-1560708537-f5debb2b4b08?w=800 | Beautiful main area

RULES FOR EACH COLUMN:
───────────────────────────────────────────────────────────────────────────────
A: Title
   - Short name for the image
   - Examples: "Salon Interior", "Styling Station", "Hair Styling"
   - Keep to 1-3 words

B: Image URL
   - REQUIRED (must have)
   - COMPLETE URL starting with https://
   - Image must be publicly accessible
   - Must work in browser

C: Description
   - Optional but recommended
   - Short description (2-5 words)
   - Examples: "Beautiful salon area", "Professional station"
   - Can be left empty

REQUIRED MINIMUM:
  - At least 3 images
  - Column B (Image URL) MUST be filled
  - Column A (Title) MUST be filled

================================================================================
                        STEP-BY-STEP SETUP
================================================================================

1. GO TO GOOGLE SHEETS
   → sheets.google.com
   → Click "+ Create"
   → "Blank spreadsheet"

2. NAME YOUR SHEET
   → "Salon Services & Gallery" (or your name)

3. CREATE THREE SHEET TABS
   → Right-click "Sheet1" → Rename to "Services"
   → Click "+" → New sheet → Rename to "Gallery"
   → Click "+" → New sheet → Rename to "Settings"

4. FILL IN DATA
   → Services sheet: Add headers + 6+ services
   → Gallery sheet: Add headers + 3+ images
   → Settings sheet: Add headers + 8 settings

5. MAKE PUBLIC (CRITICAL!)
   → Click "Share" button
   → Change to "Anyone with the link"
   → Select "Viewer" access
   → Click "Copy link"

6. COPY YOUR SHEET ID
   → Look at URL: docs.google.com/spreadsheets/d/{SHEET_ID}/edit
   → Copy the {SHEET_ID} part

7. SHARE WITH DEVELOPER
   → SHEET_ID: [your sheet ID]
   → Public Link: [share link from step 5]

================================================================================
                        COMMON MISTAKES
================================================================================

❌ WRONG:
   Price column: "$65", "65.00", "65$"
   Duration column: "60 min", "1 hour", "60 minutes"
   Category: "haircut", "HAIR", "hair service"
   Image URL: "/image.jpg", "image.jpg", "www.example.com"

✅ RIGHT:
   Price column: 65
   Duration column: 60
   Category: Hair
   Image URL: https://example.com/image.jpg

❌ Sheet is PRIVATE (not shared)
✅ Sheet is PUBLIC (shared with "Viewer" access)

❌ Column headers misspelled or in wrong order
✅ Headers exactly match the format shown

❌ Empty rows in the middle of data
✅ Data is continuous without gaps

================================================================================
                        TROUBLESHOOTING
================================================================================

PROBLEM: Website shows old/demo data, not my Google Sheets
SOLUTION:
  1. Is sheet PUBLIC? (most common issue!)
  2. Check sheet names: "Services", "Gallery", "Settings" (exact spelling)
  3. Check headers match exactly
  4. No empty rows in data
  5. Phone developer with SHEET_ID

PROBLEM: Prices/duration show blank on website
SOLUTION:
  1. Check Price and Duration are NUMBERS ONLY
  2. No "$" in price, no "min" in duration
  3. Check column order: Name, Price, Duration, Category, Description, Image
  4. No spaces or special characters in numbers

PROBLEM: Gallery images don't appear
SOLUTION:
  1. Image URL must be COMPLETE (start with https://)
  2. Try opening URL in new tab - image should load
  3. Check it's a publicly accessible image
  4. Try image from Unsplash instead

PROBLEM: Special characters appear as ?, %, &
SOLUTION:
  1. Make sure Sheet is set to UTF-8 encoding
  2. Try retyping the text
  3. Avoid special characters, use plain text

PROBLEM: Category filter shows empty options
SOLUTION:
  1. Check Category values match exactly: Hair, Skincare, Nails, Wellness, Makeup
  2. No typos or extra spaces
  3. Check capitalization matches exactly
