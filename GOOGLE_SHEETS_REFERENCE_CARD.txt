================================================================================
                        SETUP CHECKLIST
================================================================================

PART 1: CREATE SHEET & TABS
  ☐ Go to sheets.google.com
  ☐ Click "+ Create" → "Blank spreadsheet"
  ☐ Name it (e.g., "Salon Data")
  ☐ Rename "Sheet1" to "Services"
  ☐ Add sheet → Rename to "Gallery"
  ☐ Add sheet → Rename to "Settings"

PART 2: SERVICES SHEET
  ☐ Add headers in Row 1
  ☐ Add 6+ services in Rows 2+
  ☐ All prices are numbers
  ☐ All durations are numbers
  ☐ All categories match exactly
  ☐ All descriptions are complete
  ☐ Image URLs are complete (or empty)

PART 3: GALLERY SHEET
  ☐ Add headers in Row 1
  ☐ Add 3+ images in Rows 2+
  ☐ All Image URLs are filled
  ☐ All Image URLs start with https://
  ☐ All Image URLs work (test in browser)

PART 4: SETTINGS SHEET
  ☐ Add headers in Row 1
  ☐ Add all 8 settings in Rows 2-9
  ☐ Setting names are spelled exactly right
  ☐ All values are filled in

PART 5: SHARE & VERIFY
  ☐ Click "Share" button
  ☐ Change to "Anyone with the link"
  ☐ Select "Viewer" access
  ☐ Copy share link
  ☐ Test in incognito mode (open share link)
  ☐ Can see data without logging in?

PART 6: PREPARE FOR DEVELOPER
  ☐ Copy SHEET_ID from URL
  ☐ Save SHEET_ID
  ☐ Save public share link

================================================================================
                       COMMON MISTAKES & FIXES
================================================================================

WRONG → RIGHT

Price: "$65"           → Price: 65
Price: "65.00"        → Price: 65
Duration: "60 min"    → Duration: 60
Duration: "1 hour"    → Duration: 60
Category: "haircut"   → Category: Hair
Category: "HAIR"      → Category: Hair
Image: "/photo.jpg"   → Image: https://...jpg
Image: "photo.jpg"    → Image: https://...jpg

Setting: "phone"      → Setting: Phone
Setting: "Email"      → Setting: Email
Setting: "whatsapp"   → Setting: WhatsApp

================================================================================
                      TROUBLESHOOTING
================================================================================

PROBLEM: Website shows demo/default data, not my Google Sheets
  → Is sheet PUBLIC? (MOST COMMON!)
  → Solution: Share → "Viewer" access → Share

PROBLEM: Prices appear blank on website
  → Contains $ or decimal? (65.00, $65, etc.)
  → Solution: Use NUMBERS ONLY (65)

PROBLEM: Gallery images don't show
  → Image URL incomplete or broken?
  → Solution: Use complete https://... URL
  → Test: Paste URL in browser, image should load

PROBLEM: Services won't filter by category
  → Category misspelled or wrong case?
  → Solution: Use EXACTLY: Hair, Skincare, Nails, Wellness, Makeup

PROBLEM: Settings don't appear on website
  → Setting names misspelled?
  → Solution: Check exact spelling in SETTINGS SHEET RULES above

PROBLEM: Can't find SHEET_ID
  → Look at URL: docs.google.com/spreadsheets/d/{SHEET_ID}/edit
  → Copy the {SHEET_ID} part
